# Rock Paper Scissors Game

A simple Rock Paper Scissors game built using **HTML, CSS, and JavaScript**.

## Features

- Play against the computer
- Random computer choice
- Live score updates
- Win, Lose, and Draw messages
- Simple and responsive design

## Technologies Used

- HTML
- CSS
- JavaScript

style.css
